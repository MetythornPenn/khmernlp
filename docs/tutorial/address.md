# Address Module

The `address` module provides functionalities to list and filter address data in Khmer.

## Functions

### km_villages(province='all')

List all villages.

**Arguments:**
- `province` (str): The province name to filter by. If 'all', list all data.

**Returns:**
- List of villages.

### km_commune(province='all')

List all communes.

**Arguments:**
- `province` (str): The province name to filter by. If 'all', list all data.

**Returns:**
- List of communes.

### km_districts(province='all')

List all districts.

**Arguments:**
- `province` (str): The province name to filter by. If 'all', list all data.

**Returns:**
- List of districts.

### km_provinces()

List all provinces.

**Returns:**
- List of provinces.
